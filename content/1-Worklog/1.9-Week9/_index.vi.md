* Phát triển mã nguồn các hàm AWS Lambda xử lý logic nghiệp vụ và tích hợp Amazon DynamoDB/S3.

### Kết quả đạt được tuần 9:

* Hoàn thiện mã nguồn các hàm **AWS Lambda** xử lý logic nghiệp vụ của hệ thống.
* Tích hợp thành công Lambda với **Amazon DynamoDB/S3** để lưu trữ dữ liệu đơn hàng.
* Cấu hình thành công **Amazon API Gateway** ở cả hai dạng:
  * REST API (tiếp nhận request từ client)
  * WebSocket API (cập nhật trạng thái thời gian thực)
* Triển khai và hoàn thành kiểm thử tích hợp (End-to-End) cho toàn bộ luồng nghiệp vụ.
* Tối ưu hóa được thời gian phản hồi (Latency) của hệ thống.
* Biết cách rà soát, phát hiện lỗi hệ thống thông qua **CloudWatch**.
